LOCK OBJECT OFF
This command will unlock the specified 3D object from the screen. Locking objects to the screen commands the object to completely ignore the cameras influence. By unlocking a previously locked object, you are commanding the object to associate itself with the influence of the camera once again. The Object Number should be specified using an integer value.

SYNTAX
LOCK OBJECT OFF Object Number
