NET PLAYER CREATED
This command will return the Player Number of a newly created player to the net game. The Player Number is the Unique ID you can use to initialise a new player in your game data. You do not need to create a new player when you recieve this signal as it has already been done for you.

SYNTAX
Return Value = NET PLAYER CREATED()
