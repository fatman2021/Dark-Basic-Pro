CASE DEFAULT
Use this command in combination with SELECT, ENDCASE and ENDSELECT to create a select statement. A case default statment holds code below it in the event that no case statement catches the value of the variable. The case default statment must go at the end of a sequence of case statement, but above the ENDSELECT statement.

SYNTAX
