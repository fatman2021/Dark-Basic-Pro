LIGHT 3D COMMAND SET

3D Lights are used to illuminate your 3D scenes, creating anything from subtle ambient effects, to bright cones of light and shadows. You are able to create upto 7 unique sources of light, each with its own style, colour, position and in some cases direction. You can also modify light zero, which is the default illumination for an empty scene.

Click on any commands for more information:

LIGHT COMMANDS

MAKE LIGHT will create a new light source
DELETE LIGHT will delete an existing light source
SET POINT LIGHT will set the light source to be omnidirectional
SET SPOT LIGHT will set the light source to be cone shaped
SET DIRECTIONAL LIGHT will set the light source in one direction
SET LIGHT RANGE This command will set the range of the light will shine
POSITION LIGHT will position the light source
ROTATE LIGHT will rotate the orientation of the light source
POINT LIGHT will aim the light source at a location
COLOR LIGHT will define the color of the light source
HIDE LIGHT will disable the light source
SHOW LIGHT will enable the light source
SET LIGHT TO OBJECT POSITION will position the light to an object
SET LIGHT TO OBJECT ORIENTATION will align the light to an object
SET AMBIENT LIGHT will set the percentage level of ambient light
COLOR AMBIENT LIGHT changes the current ambient light colour

FOG COMMANDS

FOG ON will activate the effect of fogging
FOG COLOR will set the color of the fogging effect
FOG DISTANCE will set the visible distance of the fog
FOG OFF will deactivate the effect of fogging

The next commands will return a result:

LIGHT EXIST() returns a one if the light exists
LIGHT TYPE() returns 1 for point, 2 for spot and 3 for directional light type
LIGHT POSITION X() returns the X position of the light
LIGHT POSITION Y() returns the Y position of the light
LIGHT POSITION Z() returns the Z position of the light
LIGHT DIRECTION X() returns the X value of the light direction
LIGHT DIRECTION Y() returns the Y value of the light direction
LIGHT DIRECTION Z() returns the Z value of the light direction
LIGHT VISIBLE() returns a one if the light is active
LIGHT RANGE() returns the current range of the light source
