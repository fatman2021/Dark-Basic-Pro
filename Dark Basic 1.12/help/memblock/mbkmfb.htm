MAKE MEMBLOCK FROM BITMAP
This command will make a memblock from a bitmap. The specified values must be integer values and the source resource must exist or the command will fail. The memblock stores the data as described below.

Memblock-Bitmap (12 Byte Header)
-Header
--4 BYTES (DWORD) Width
--4 BYTES (DWORD) Height
--4 BYTES (DWORD) Depth
-Data
--X BYTES - Pixel Data (a 16bit bitmap uses 2 BYTES per Pixel)

SYNTAX
MAKE MEMBLOCK FROM BITMAP Memblock Number, Bitmap Number
