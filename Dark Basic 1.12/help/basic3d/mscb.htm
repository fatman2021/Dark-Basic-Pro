MAKE STATIC COLLISION BOX
This command will create an invisible collision zones within the static world data. Once defined, you can use the GET STATIC COLLISION HIT command to detect whether another dynamically specified box area collides with this one. This command is ideally used to define solid areas within your 3D world.

SYNTAX
MAKE STATIC COLLISION BOX x1,y1,z1,x2,y2,z2
