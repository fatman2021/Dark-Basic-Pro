3D Tutorials: 2. The Matrix
1. Creating a matrix

Purpose:  To Introduce the matrix commands

    Rem Setup sync
    Sync On
    Sync Rate 30

    Rem Create matrix
    Make Matrix 1,2000,2000,50,50

    Rem mY is the variable for random matrix height
    mY=100

    Rem Position Camera above the matrix
    Position Camera 0,1000,0

    Rem Main loop
    Do

         Rem  Control input for camera
         If Upkey()=1 then Move camera 10
         If Downkey()=1 then Move camera -10

         Rem input for radomizing the matrix
         If Spacekey()=1 then randomize matrix 1,mY
         If Leftkey()=1 then mY=mY+1
         If Rightkey()=1 then mY=mY-1
         If mY < 1 then mY =1

         Rem Point camera at matrix center
         Point Camera 1000,0,1000

         Rem Refresh Screen
         Sync
    Loop
 
 

In this example we will introduce the matrix commands. A matrix finds its most popular use as a terrain object. You can easily create mountains, rolling hills, caves and oceans with a few lines of code. As with all of the tutorials, if you see a command that is unfamiliar and not explained, review earlier tutorials or consult the help files. This simple program creates a matrix and allows you to zoom in and out in isometric view using the up and down keys. The up, down and spacebar keys allow you to control the height and radomization of the matrix.

    Make Matrix 1,2000,2000,50,50

The first new command we will discuss is the "Make Matrix" command. This command creates a matrix object in your 3D world. The first number after the command is the number of the matrix you wish to create. This number like the object and image commands is a unique identifier for the matrix you wish to create. It allows you to create and manipulate multiple matrices in the 3D world. The second two numbers are the dimensions in world units of the matrix you wish to create. They represent an "X" and "Z" value in world coordinates. In most cases these two will be the same but we're sure you can find uses for odd sized matrices. The second two numbers are the number of divisions you wish to have in the matrix. In this example 50 divisions of 40 world units are made in each directions to create a 2000x2000 world unit matrix.
 

    If Spacekey()=1 then Randomize Matrix 1,mY

This line of code introduces the "Randomize Matrix" command". When the space key is pressed this command is called to set the height value of each of the matrix vertices to a random height between 0 and the number represented by the variable "mY". The first number after the command is the number of the matrix we wish to affect. The second number is the range of the random value we wish to set the matrix to. In the program you can change the value of the "mY" variable by pressing the up and down keys. When you press the space key the matrix is changed. This allows you to create sharp peaks or rolling hills.

Try changing the values in the "Make Matrix" command to see how they effect the size and dimensions of the matrix.
