3D Tutorials: 1. The Basics
3. Moving the camera

Purpose: To show how to change the position and rotation of the camera using keyboard input.
 

    Rem Setup sync
      Sync On
      Sync Rate 30

    Rem Create 5 cubes and place randomly
      For x = 1 to 5
        Make object cube x,100
        Position object x,Rnd(2000),0,Rnd(2000)
      Next x

    Rem Main loop
      Do

      Rem  Store camera angle
        caY#= Camera angle Y()

      Rem  Control input for camera movement
        If Upkey()=1 then Move camera 10
        If Leftkey()=1 then Yrotate Camera Wrapvalue(caY#-5)
        If Rightkey()=1 then Yrotate Camera Wrapvalue(caY#+5)

      Rem Refresh Screen
        Sync

      Loop

We will begin with the introduction of the "For Next" loop.

      For x = 1 to 5
        Make object cube x,100
        Position object x,Rnd(2000),0,Rnd(2000)
      Next x

The "For Next" loop is used to run a command, or several commands, a set number of times. We use the variable "x" in this loop to store the count of the number of times the loop will be executed. In the "For Next" loop you always use the word "To" to specify the number of times you wish to execute the loop. In this case we execute the loop from 1 "To" 5 times. We place the commands we wish to execute in the body of the loop.  We use "Next" and the counting variable "x" to signify the end of the loop and to increment the value of "x". Inside the loop we create our cube objects, giving them an object number with the same variable we use in the loop. Next, we use the "Position Object" command to position the object at a 3D coordinate in our scene. The first number in the "Position Object" command is the number of the object we wish to position. We have replaced this number with the "x" variable. The three values after the object number are the X,Y, and Z Coordinates where we wish to position the object. In place of the X and Z coordinates we have placed the "Rnd" command. The "Rnd" command is used to create a random number between 0 and the number you place in the parenthesis. Each time you run the "Position Object" command the new object will be positioned at a random X and Z coordinate and a Y coordinate of 0.

      If Upkey()=1 then Move camera 10

Our next new command is the "Move Camera" command. This command is similar to the "Move Object" command, except you don't have to specify an object number. The number after the command is the number of world units you wish to move the camera when the up key is pressed in each iteration of the main loop.

      If Leftkey()=1 then Yrotate Camera Wrapvalue(caY#-5)
      If Rightkey()=1 then Yrotate Camera Wrapvalue(caY#+5)

The "YRotate Camera" command is the same as the "YRotate Object" command, except this command rotates the camera around the camera's Y axis.

You should now have an understanding of  how to move and position the camera. Try these changes to the original program to gain a further understanding of the commands.

Try changing the value in the "Rnd" command to see how the position of the objects change in relation to each other.

Try changing the values in the "Move Camera" and "Yrotate Camera" commands to see how fast and slow you can move and rotate the camera.
