PERFORM CHECKLIST FOR CONTROL DEVICES
This command will make a checklist of all installed control devices. Control devices can range from a force-feedback joystick to head mounted displays. You can read the name of each found device by reading the Checklist String after using this command. Additionally, you can determine whether the control device has forcefeedback capability by checking for a value of 1 in Checklist Value A. In general, most devices simply provide input to the program in a standard X, Y and Z axis format.

SYNTAX
PERFORM CHECKLIST FOR CONTROL DEVICES
