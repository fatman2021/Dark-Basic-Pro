NET MESSAGE MEMBLOCK
This command will return a memblock from the current net message. The net message must be of the memblock type or the command will fail. You can determine type using the NET MESSAGE TYPE command. If a memblock already exists under this number, that memblock is deleted and the new memblock is created in its place.

SYNTAX
NET MESSAGE MEMBLOCK Memblock Number
