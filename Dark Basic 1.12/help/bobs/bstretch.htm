STRETCH BOB
This command will expand or shrinks the specified bob according to the scale values provided. You must specify both a horizontal and vertical scale when stretching bobs. If the scale value is zero, the bob will disappear. If the scale value is 100, the bob will be set to its original size. If the scale value is set to 200, the size of the bob will double. The parameters should be specified using integer values.

SYNTAX
STRETCH BOB Bob Number, X-Scale Value, Y-Scale Value
