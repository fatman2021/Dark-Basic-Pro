LOAD MESH
This command will load a single mesh file into the specified mesh number. A mesh is a wireframe description of a 3D shape. You must use a filename that points to a file that stores 3D mesh data in the X file format. The mesh number should be specified using an integer value.

SYNTAX
LOAD MESH Filename, Mesh Number
