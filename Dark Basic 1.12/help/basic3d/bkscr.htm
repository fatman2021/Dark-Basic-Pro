SCROLL BACKDROP
This command will scroll the 3D backdrop using the specified X and Y scroll values. The values represent the percentage used to scroll the textures used by the backdrop. The textures are pasted onto the backdrop as two rows of 3 tiles, which create six visible textures that make up the backdrop. An X value of 50 will scroll the texture across by exactly half, and the result of an X value of 100 is identical to an X value of zero. The X and Y values should be specified using integer values.

SYNTAX
SCROLL BACKDROP X Value, Y Value
