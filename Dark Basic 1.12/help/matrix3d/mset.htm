SET MATRIX
This command will change the visual properties of the matrix. When the wireframe flag is set to 0, the matrix only shows it's wireframe form. When the transparency flag is set to 0, all parts of the matrix colored black are not drawn to the screen. When the cull flag is set to 0, the matrix will draw polygons normally hidden due to the direction the polygon faces. The Filter Flag activates and deactivates texture filtering, which controls the smoothing effect of the texture as it is mapped to the matrix. The Light Flag activates and deactivates the matrices sensitivity to any lights in the scene. The Fog Flag activates and deactivates the matrices sensitivity to fog in the scene. The Ambient Flag activates and deactivates the matrices sensitivity to ambient light in the scene. The matrix number and flag values should be specified using integer values.

SYNTAX
SET MATRIX Matrix Number, Wireframe, Transparency, Cull, Filter, Light, Fog, Ambient
