MULTIPLAYER COMMAND SET

Multiplayer is used to create and control games whose players reside on different machines. This is achieved through a standardised connection system which allows the same commands to be used no matter whether the game was created across a serial connection, modem, local area network or Internet. The net game commands are extremely simple to use, and you can create a game and send data in just two commands! There can only be one net game per application, however with multiple applications running you can simulate a network game on a single machine.

Click on any commands for more information:

PERFORM CHECKLIST FOR NET CONNECTIONS will create list of connections
SET NET CONNECTION will choose a connection type to use
PERFORM CHECKLIST FOR NET SESSIONS will create list of game sessions

CREATE NET GAME will create a net game
JOIN NET GAME will join an existing net game
FREE NET GAME will free the current net game

PERFORM CHECKLIST FOR NET PLAYERS will create a list of all players
CREATE NET PLAYER will create an additional player for the net game
FREE NET PLAYER will remove a player from the net game

SEND NET MESSAGE INTEGER will send an integer value message
SEND NET MESSAGE FLOAT will send a float value message
SEND NET MESSAGE STRING will send a string message
SEND NET MESSAGE MEMBLOCK will send a memblock message
SEND NET MESSAGE IMAGE will send an image message
SEND NET MESSAGE BITMAP will send a bitmap message
SEND NET MESSAGE SOUND will send a sound message
SEND NET MESSAGE MESH will send a mesh message
GET NET MESSAGE will get the oldest incoming message

The next commands will return a result:

NET MESSAGE INTEGER() will return an integer value message
NET MESSAGE FLOAT() will return a float value message
NET MESSAGE STRING() will return a string message
NET MESSAGE MEMBLOCK() will return a memblock message
NET MESSAGE IMAGE will return an image message
NET MESSAGE BITMAP will return a bitmap message
NET MESSAGE SOUND will return a sound message
NET MESSAGE MESH will return a mesh message

NET GAME EXISTS() will return a one if the net game exists
NET GAME NOW HOSTING() will return a one if gained host status
NET GAME LOST() will return a one if net game has been lost
NET MESSAGE EXISTS() will return a one if message exists
NET MESSAGE PLAYER FROM() will return who the message is from
NET MESSAGE PLAYER TO() will return who the message is for
NET MESSAGE TYPE() will return the type of the current message
NET PLAYER DESTROYED() will return who has been destroyed
NET PLAYER CREATED() will return who has been created
