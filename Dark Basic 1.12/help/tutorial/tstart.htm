DARK BASIC
 MAIN TUTORIALS MENU

These tutorials have been designed to walk you through each part of the language. The first five have been graded by difficulty ranging from beginner to novice:

1. HELLO WORLD is the universal BASIC program everyone should learn
2. RANDOM PRINTER randomly pastes the screen full of text
3. SCREEN SWITCHER detects for display modes and changes screen resolutions
4. ARTISTIC VARIABLES uses variables and mathematics to create art
5. STOCK MARKET ARRAYS lets you create your own stock market database

The next five tutorials all use media files and require an understanding of external files. These tutorials likewise have been graded to be incrementally more difficult:

6. PHOTO COPIER loads and copies bitmap images
7. SOUND AND MUSIC explode into waves of sounds and music
8. WALKING SPRITE makes a simple 2D character walk
9. TERRA FORM creates rolling 3D landscapes to fly over
10. 3D THAT WALKS summons a 3D character for you to control
