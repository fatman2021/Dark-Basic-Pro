BOB HIT()
This command will return a one if the specified bob has impacted against the target bob specified. If a target bob has not been specified and a value of zero have been used, this command will return the bob number of any bob impacting against it. The parameters should be specified using integer values.

SYNTAX
Return Value=BOB HIT(Bob Number, Target Bob Number)
