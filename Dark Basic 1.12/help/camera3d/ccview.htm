CLEAR CAMERA VIEW
This command will clear the viewport of the camera. The viewport is the actual area on screen where all 3D is drawn. The default viewport area is the entire screen. You can specify that only the camera viewport be cleared using this command. This area can be changed using rhe Set Camera View command. You must specify the colour value using an integer value.

SYNTAX
CLEAR CAMERA VIEW Colour Value
