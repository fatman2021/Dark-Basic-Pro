MAKE BITMAP FROM MEMBLOCK
This command will make a bitmap from a memblock. The specified values must be integer values and the source resource must exist or the command will fail. The memblock stores the data as described below.

Memblock-Bitmap (12 Byte Header)
-Header
--4 BYTES (DWORD) Width
--4 BYTES (DWORD) Height
--4 BYTES (DWORD) Depth
-Data
--X BYTES - Pixel Data (a 16bit bitmap uses 2 BYTES per Pixel)

SYNTAX
MAKE BITMAP FROM MEMBLOCK Bitmap Number, Memblock Number
