COULD NOT RECOGNIZE COMMAND UNDER CURSOR

You have reached this page because you tried to use context sensitive help but did not indicate what you wanted help with.

In order to use context sensitive help, simply place the flashing text cursor (the caret) anywhere within the text of the command you are interested in and press F1.

If the command is recognized, you will be taken to the help page that describes what the command does, the syntax you should use and what each parameter requires. You will also be given links to the glossary that explains terminology used to describing the command. All help pages provide you with a link to a working example that demonstrates the command being used.
