SCREEN COMMAND SET

The screen can be set to any mode your graphics card supports excluding 8bit display modes (256 colours). The default display mode is 640 x 480 x 16bit and is the most common resolution to use. You are only limited by the speed, capacity and capability of your graphics card when setting your display mode. Avoid setting a display mode that consumes most of your available video memory as this will impair 3D performance. Avoid setting a display mode that exceeds your available video memory as this will dramatically slow down your screen refresh rate.

Click on any commands for more information:

PERFORM CHECKLIST FOR DISPLAY MODES will list all display modes you can use
SET DISPLAY MODE will set the screen display mode provided it is available
SET GAMMA will set the screens red, green and blue gamma levels

The next commands will return a result:

CHECK DISPLAY MODE() will return a one if the display mode is available
SCREEN TYPE() will return the current screen type
SCREEN WIDTH() will return the current screen width
SCREEN HEIGHT() will return the current screen height
SCREEN DEPTH() will return the current screen depth
SCREEN FPS() will return the current frames per second
