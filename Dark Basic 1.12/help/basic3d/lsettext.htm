TEXTURE LIMB
This command will apply an existing image to the limb of a 3D object as a texture. You must create an image first using the GET IMAGE command before attempting to texture part of the 3D object. The parameters should be specified using integer values.

The image for the texture can be any size, but the quality of the texture depends on the graphics card used. Most graphics cards support a texture size of 256x256. This is the recommended size for all your textures. Where an image is larger than the supported texture size, the image is internally reduced when used as a texture.

SYNTAX
TEXTURE LIMB Object Number, Limb Number, Image Number
