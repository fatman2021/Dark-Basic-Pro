HIDE OBJECT
This command will hide the specified 3D object from view. You can substantially increase the performance of your 3D program if you hide objects when ever possible. The contents of a room behind a closed door can be hidden for as long as the door remains closed, allowing your program to run much faster and improve overall performance. The parameter should be specified using an integer value.

SYNTAX
HIDE OBJECT Object Number
