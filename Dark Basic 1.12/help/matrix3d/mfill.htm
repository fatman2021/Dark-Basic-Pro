FILL MATRIX
This command will set each grid square in the matrix to a specified height and tile number. The matrix number should be an integer value. The height should be a real number. The tile number must be valid texture tile allocated by the PREPARE MATRIX TEXTURE command and should be an integer value.

SYNTAX
FILL MATRIX Matrix Number, Height, Tile Number
