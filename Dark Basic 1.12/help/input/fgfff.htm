FTP GET FILE
This command will allow you to use FTP PROCEED to grab. The GrabInBits Flag allows you to specify the amount in bytes to be grabbed each time FTO PROCEED is called, thus controlling the rate and responsiveness of your application during download.

SYNTAX
FTP GET FILE Ftp Filename String, Local filename String, GrabInBits Flag
