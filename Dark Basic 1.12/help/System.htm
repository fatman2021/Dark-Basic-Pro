SYSTEM COMMAND SET

Your PC, and indeed any PC system that runs your software will have a wide range of capabilities that your program may wish to detect and utilize. You may also wish to present the user with a selection of possible modes and allow a selection to be made.

Click on any commands for more information:

PERFORM CHECKLIST FOR GRAPHICS CARDS will make a list of all display cards
SET GRAPHICS CARD will set the current display card
SET EMULATION ON will select software emulation
SET EMULATION OFF will select hardware acceleration
EMPTY CHECKLIST will clear the general purpose checklist facility

DISABLE ESCAPEKEY will disable the escape key exit
ENABLE ESCAPEKEY will enable the escape key exit
DISABLE TNL will disable the use of hardware TnL
ENABLE TNL will enable the use of hardware TnL

HIDE WINDOW will hide the window but keeps DB active
SHOW WINDOW will show the window
MINIMIZE WINDOW will minimise the window to a desktop iconized bar
MAXIMIZE WINDOW will maximise the window to the entire desktop screen
RESTORE WINDOW will restore the window top the original size and position
ALWAYS ACTIVE ON will set the application to stay active when minimized
ALWAYS ACTIVE OFF set restore the application to normal active state

SET WINDOW TITLE will change the title of the window
SET WINDOW ON will activate Windows Mode
SET WINDOW OFF will deactivate Windows Mode
SET WINDOW LAYOUT will set window settings
SET WINDOW POSITION will set the windows position
SET WINDOW SIZE will set the windows size

DISABLE SYSTEMKEYS will disable use of such system keys as ALT+TAB, ALT+F4
ENABLE SYSTEMKEYS will enable all system keys disabled with DISABLE SYSTEMKEYS

LOAD DLL will load a DLL into memory
DELETE DLL will delete a DLL from memory
CALL DLL will call the function of a loaded DLL

EXIT PROMPT will create message box with a specified string on termination

The next commands will return a result:

CHECKLIST QUANTITY() will return the total number of items in the checklist
CHECKLIST VALUE A() will return value A from the specified item in the checklist
CHECKLIST VALUE B() will return value B from the specified item in the checklist
CHECKLIST VALUE C() will return value C from the specified item in the checklist
CHECKLIST VALUE D() will return value D from the specified item in the checklist
CHECKLIST STRING$() will return the string from the specified item in the checklist

SCREEN INVALID() returns a one if the application has been switched out and back in

CURRENT GRAPHICS CARD$() will return the name of the current display card
SYSTEM DMEM AVAILABLE() will return the total video memory available
SYSTEM SMEM AVAILABLE() will return the total system memory available
SYSTEM TMEM AVAILABLE() will return the overall system memory available
EMULATION MODE() will return a one if the system is in software emulation mode
TNL AVAILABLE() will return a one if TnL in hardware is being used

DLL EXIST() will return a one if the DLL has been loaded
DLL CALL EXIST() will return a one if the DLL function exists
