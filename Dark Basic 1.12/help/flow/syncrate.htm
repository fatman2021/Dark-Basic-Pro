SYNC RATE
This command is used to change the default refresh rate SYNC ON uses to control the speed of the SYNC update speed. The default rate sustains the 'Frames Per Second' at no more than 40fps. You can specify an integer value from 1 to 1000 to set a new maximum rate. A rate of zero will allow the program to refresh as fast as the system will allow.

SYNTAX
SYNC RATE Refresh Rate
