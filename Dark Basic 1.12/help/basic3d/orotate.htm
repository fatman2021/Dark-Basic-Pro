ROTATE OBJECT
This command will rotate the specified 3D object around all three dimensions. The object number should be specified using an integer value. This method of rotation is called euler rotation and differs from free flight rotation. Euler angles retrieved from a free flight rotation specify a ZYX rotation, and you must call the SET OBJECT ROTATION ZYX command on the Euler based object to make the two rotation systems compatable. The rotation angles should be specified using real numbers.

SYNTAX
ROTATE OBJECT Object Number, X, Y, Z
