SCALE BOB
This command will expand or shrinks the specified bob according to the scale value provided. If the scale value is zero, the bob will disappear. If the scale value is 100, the bob will be set to its original size. If the scale value is set to 200, the size of the bob will double. The parameters should be specified using integer values.

SYNTAX
SCALE BOB Bob Number, Scale Value
