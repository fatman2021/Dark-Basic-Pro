LOAD IMAGE
This command will load a bitmap file as an image. You can optionally specify a texturing flag which pre-creates the image as a texture for later use on a 3D object. Pre-creating the image as a texture speeds up the initial texturing of any 3D object. You must specify an image number between 1 and 65535.

SYNTAX
LOAD IMAGE Filename String, Image Number
LOAD IMAGE Filename String, Image Number, Texturing Flag
