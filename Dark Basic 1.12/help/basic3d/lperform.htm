PERFORM CHECKLIST FOR OBJECT LIMBS
This command will make a list of all limbs contained in a 3D object. Objects that have been loaded may contain hundreds of limbs, and are identified by number. All limbs include an internal limb description that often indicates which part of the overall 3D object it belongs to. You can access the limb description using the string item of the checklist when you have performed the check.

SYNTAX
PERFORM CHECKLIST FOR OBJECT LIMBS
