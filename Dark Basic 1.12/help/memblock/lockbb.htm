LOCK BACKBUFFER
This command locks the backbuffer and stores the backbuffer details for direct access. The backbuffer is the actual screen you see each refresh and this command allows you to pass the pointer and other essential data to a DLL for screen modification. Locking the backbuffer prevents other activities to be performed regarding the backbuffer so it is essential you unlock the backbuffer when you have finished.

SYNTAX
LOCK BACKBUFFER
