MEMBLOCK COMMAND SET

Memblocks are used to store and manipulate various types of data. Unlike arrays that can store basic values, memory blocks can be used to manipulate bitmap, image, sound and even 3D mesh data. There are 256 memblocks available and share the common ability to manipulate data at the byte level, share their internal memory pointers and copy between memblocks easily.

Click on any commands for more information:

MAKE MEMBLOCK will make a memblock of a given size
DELETE MEMBLOCK will delete an existing memblock
COPY MEMBLOCK will copy part of one memblock to another memblock
WRITE MEMBLOCK BYTE will write a single byte to a memblock
WRITE MEMBLOCK WORD will write a single word to a memblock
WRITE MEMBLOCK DWORD will write a single dword to a memblock
WRITE MEMBLOCK FLOAT will write a single float to a memblock

MAKE MEMBLOCK FROM BITMAP will make a memblock from a bitmap
MAKE BITMAP FROM MEMBLOCK will make a bitmap from a memblock
MAKE MEMBLOCK FROM IMAGE will make a memblock from an image
MAKE IMAGE FROM MEMBLOCK will make an image from a memblock
MAKE MEMBLOCK FROM SOUND will make a memblock from a sound
MAKE SOUND FROM MEMBLOCK will make a sound from a memblock
MAKE MEMBLOCK FROM MESH will make a memblock from a mesh
MAKE MESH FROM MEMBLOCK will make a mesh from a memblock
CHANGE MESH FROM MEMBLOCK will update a mesh from a memblock

LOCK BACKBUFFER will lock the backbuffer for direct access
UNLOCK BACKBUFFER will unlock the backbuffer after a lock

The next commands will return a result:

MEMBLOCK EXIST will return a one if the memblock exists
GET MEMBLOCK PTR will return the pointer to the memblock
GET MEMBLOCK SIZE will return the size of the memblock
MEMBLOCK BYTE will return a single byte from the memblock
MEMBLOCK WORD will return a single word from the memblock
MEMBLOCK DWORD will return a single dword from the memblock
MEMBLOCK FLOAT will return a single float from the memblock

GET BACKBUFFER PTR will get the pointer of a locked backbuffer
GET BACKBUFFER WIDTH will get the width of a locked backbuffer
GET BACKBUFFER HEIGHT will get the height of a locked backbuffer
GET BACKBUFFER DEPTH will get the depth of a locked backbuffer
GET BACKBUFFER PITCH will get the pitch of a locked backbuffer
