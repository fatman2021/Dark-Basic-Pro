JOYSTICK HAT ANGLE()
This command will give support for reading upto 4 HAT controllers. When the hat controller is neutral, a value of -1 is returned. When the hat is being used, the return value is in tens of a degree so north would be zero, east would be 900, south is 1800 and west is 2700.

SYNTAX
Return Value=JOYSTICK HAT ANGLE(Hat Number)
