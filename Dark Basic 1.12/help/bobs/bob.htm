BOB
This command will set the position and image number of the specified bob. Providing you are using a valid image number from a previous call to the GET IMAGE command, and the position of the bob is in the screen area, you will see your bob displayed. You can move your bob by calling the bob command with new position coordinates. You can animate your bob by calling this command with different image numbers to create the effect of animation. You are able to have over sixty thousand bobs on the screen at any one time, but it is advisable to restrict yourself to a few hundred bobs for speed critical programs. The parameters should be specified using integer values.

SYNTAX
BOB Bob Number, X, Y, Image Number
