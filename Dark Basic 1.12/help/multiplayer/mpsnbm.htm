SEND NET MESSAGE BITMAP
This command will send a message containing a bitmap to the specified player. The Player Number must be an integer value and an existing player in the net game. A Player Number of zero will send the message to all players except you. The Guarentee Packet Flag, if set to one, will ensure the message is received and will not be dropped due to slow system performance.

SYNTAX
SEND NET MESSAGE BITMAP Player Number, Bitmap Number, Guarentee Packet
