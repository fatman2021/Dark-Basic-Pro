SET SOUND PAN
This command will set the pan of standard sounds by locating it between the left and right speakers. A negative value will move the sound to the left speaker, a positive value will move it to the right. Sound panning does not work with 3D sounds. The pan value must be an integer value between -10000 and 10000.

SYNTAX
SET SOUND PAN Sound Number, Pan Value
